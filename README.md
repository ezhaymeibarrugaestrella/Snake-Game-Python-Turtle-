# Snake-Game-Python-Turtle-
A beginner-friendly Snake Game developed in Python using the Turtle module. It includes real-time movement, food spawning, score tracking, and a game-over menu with restart and exit buttons. Built for learning basic game development and OOP concepts in Python.


# 🐍 Snake Game (Python Turtle)

A simple Snake Game built using Python's built-in Turtle graphics module.  
The game includes scoring, collision detection, and a restart/exit game-over screen.

---

## 🎮 Features

- 🐍 Classic snake movement (Up, Down, Left, Right controls)
- 🍎 Random food spawning
- 📈 Score system
- 💥 Collision detection (wall & self)
- 🧑 Player name input
- 🎮 Game Over screen with:
  - Try Again button
  - Exit button

---

## 🛠️ Technologies Used

- Python 3
- Turtle Graphics (built-in library)
- Random module
- Time module

> ⚠️ No external libraries required

---

## 🎮 Controls

| Key | Action |
|-----|--------|
| ↑ Arrow | Move Up |
| ↓ Arrow | Move Down |
| ← Arrow | Move Left |
| → Arrow | Move Right |
| ENTER | Start Game |
